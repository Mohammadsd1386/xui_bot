# 🤖 ربات فروش و مدیریت VPN

## امکانات
- خرید سرویس (پلن‌های متنوع)
- پرداخت: زرین‌پال / تتر BEP20 / ترون TRC20 / تون کوین / کیف پول
- پشتیبانی از پنل سنایی (3x-ui) و مرزبان
- سیستم رفرال با پاداش خودکار
- تست رایگان قابل تنظیم
- پنل ادمین کامل از داخل ربات
- گزارش فروش به تفکیک درگاه
- مدیریت کاربران (تخفیف / مسدود / شارژ کیف)
- سیستم تیکت پشتیبانی
- ارسال همگانی

## راه‌اندازی

### ۱. فایل .env را پر کنید
```
BOT_TOKEN=توکن_ربات
OWNER_ID=آیدی_عددی_شما
SETUP_TOKEN=یک_رمز_دلخواه
```

### ۲. نصب
```bash
chmod +x install.sh && bash install.sh
```

### ۳. اجرا
```bash
source venv/bin/activate
python bot.py
```

### ۴. اجرا به‌عنوان سرویس
```bash
chmod +x setup_service.sh && sudo bash setup_service.sh
```

### ۵. در تلگرام
```
/setup your_setup_token
```
بعد از این، شما مالک ربات هستید و می‌توانید از پنل ادمین همه چیز را تنظیم کنید.

## تنظیمات از پنل ادمین
همه تنظیمات (نرخ دلار، آدرس کیف‌ها، پلن‌ها، پنل‌ها) از داخل ربات قابل تغییر است.
